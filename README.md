# Smart Attendance System

A real-time **Face Recognition Attendance System** built with React.js and Python InsightFace.

## 🚀 Features

- 📹 Live face recognition using InsightFace
- 📊 Real-time attendance dashboard (React)
- ☁️ Cloud storage integration
- 📥 Export attendance to CSV
- 📈 Recognition rate statistics

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React.js, Bootstrap 5, FontAwesome |
| Face Recognition | Python, InsightFace |
| Styling | Custom CSS, React-Bootstrap |

## 📁 Project Structure

```
smart-attendance-system/
├── react-dashboard/        ← React frontend dashboard
│   ├── src/
│   │   ├── App.js          ← Main dashboard component
│   │   └── App.css         ← Dashboard styles
│   └── package.json
├── insightface/            ← Face recognition backend
└── README.md
```

## ⚙️ Setup & Run

### Frontend (React Dashboard)
```bash
cd react-dashboard
npm install
npm start
```
Dashboard runs at: http://localhost:3000

### Models
> ⚠️ ML model files are large (700MB+) and not included in this repo.
> Download models separately and place in the `models/` folder.
